# Disk Health Lite

Portable Windows disk health monitor.

## Run

Open `DiskHealthLite.exe`.

The app asks for Administrator permission so it can read SMART and Windows storage counters.

## Included

- Disk health percentage
- Temperature
- Power-on time
- Lifetime writes
- Wear value
- Automatic refresh every 15 seconds
- Bundled Smartmontools `smartctl.exe`
- GitHub and Instagram footer links

Some USB adapters or disk drivers may hide SMART fields. When SMART data is exposed, the bundled `smartctl.exe` is used automatically.
